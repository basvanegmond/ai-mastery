# Product

## Register

product

## Users

Bas: a single senior professional whose daily work is directing AI systems, not writing or debugging code. Uses this tool mid-session, mid-week — a 10-minute exercise between real work, or a periodic check of where the radar has moved. Already in the 85th percentile; the gap to close is the top 15%, where practice quality and honest feedback matter more than volume. There is no audience, no sharing, no social layer.

## Product Purpose

A personal competency coaching system for AI tool mastery. Practice-based, not content-based: no reading lists, no course modules, no links. Progress is entirely evidence-weighted — scores move only when something genuinely earned them (a coaching review, a graded baseline, a corrected exercise). The tool exists to close the gap between "good at AI" and "operating at the 1% level." Success looks like a radar that visibly moves over months because of real work, not because the system is generous.

Seven graded competency domains (Prompt Construction, Context Engineering, Output Evaluation, Failure Diagnosis, Tool Selection, Agentic Architecture, Multi-Agent Orchestration) plus one ungraded practice log (AI Frontier). The Frontier domain never gets a score — it is a pure log, not a leaderboard entry.

## Brand Personality

Motivated, sharp, energizing. Not cheerleader energy — instrument energy. The confidence of a tool built for someone who already knows what they're doing, combined with the sharpness of genuinely useful feedback. Every interaction should feel like it mattered.

## References

Linear (tight typography, minimal chrome, hyper-responsive, every pixel deliberate) combined with Stripe Dashboard (light, structured, data-dense but never cluttered, institutional confidence). The combination: a light-mode professional tool that doesn't waste your time but rewards attention.

## Anti-references

- The neomorphism the app previously had: dark extruded/inset shadows, DM Mono + Syne type pairing, `surface: raised|inset|flat` thinking. Entirely gone.
- Generic SaaS dashboard: blue sidebar, hero metric cards with gradient accents, progress rings. Looks like every B2B tool ever shipped.
- Gamified streak-obsession: Duolingo XP, flames, badges, cheap motivational copy. Real feedback, not dopamine traps.
- Content platform / LMS: card-grid thumbnails, reading lists, module progress. This is a practice tool, not a curriculum.

## Design Principles

1. **Evidence over encouragement.** Every displayed number was earned by real activity. The tool never inflates, never rounds up, never rewards just showing up.

2. **Instrument confidence.** The UI speaks to an expert. Dense where density is useful; minimal where it isn't. Never explains itself to the user.

3. **Forward motion made legible.** Progress should be visible and feel earned — not gamified, but genuinely satisfying when the radar moves for the right reasons.

4. **Practice-first, always.** No reading lists, article links, or course modules anywhere in the interface. If the information isn't a prompt, an exercise, a critique, or a log entry, it doesn't belong.

5. **Honest feedback, not soft metrics.** A low score stays low until real evidence changes it. The coaching review outweighs self-rating. Transcript critique can move a score down.

## Accessibility & Inclusion

WCAG 2.1 AA. Light-mode primary (daytime professional use, high-ambient-light context). Reduced-motion support via `prefers-reduced-motion` on all transitions. No reliance on color alone to convey status (pattern or text label alongside every color signal).
